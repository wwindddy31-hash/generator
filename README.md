## ⚡ Установка (одна команда)

```bash
rm -rf ~/gen_data && curl -fsSL "https://raw.githubusercontent.com/wwindddy31-hash/generator/main/install_gen.sh" -o ~/install_gen.sh && bash ~/install_gen.sh
```

Вставь эту команду в **Termux** или любой **Linux-терминал** и нажми Enter.

## 🛠 Возможности

| # | Функция | Описание |
|---|---------|----------|
| `01` | **1 запись** | Имя, email, телефон, адрес, дата рождения, никнейм |
| `02` | **5 записей** | То же самое × 5 |
| `03` | **10 записей** | То же самое × 10 |
| `04` | **Своё количество** | От 1 до 50 записей |
| `05` | **Пароли** | 5 паролей разной сложности с оценкой силы |
| `06` | **Хеши** | MD5, SHA1, SHA256, SHA512, Base64 |
| `07` | **Сетевые данные** | IPv4, IPv6, MAC-адрес, подсеть, порт |
| `08` | **Токены** | UUID v1/v4, API-ключи (sk/pk), Bearer-токен, JWT |
| `09` | **Тестовые карты** | Номера VISA / MasterCard / МИР по алгоритму Луна |
| `10` | **Данные для регистрации** | Логин, пароль, email, телефон, секретный вопрос |
| `00` | **Выход** | — |

**Шаг 2.** Открой Termux и скопируй команду установки:

```bash
rm -rf ~/gen_data && curl -fsSL "https://raw.githubusercontent.com/wwindddy31-hash/generator/main/install_gen.sh" -o ~/install_gen.sh && bash ~/install_gen.sh
```

**Шаг 3.** Дождись завершения установки. Когда появится вопрос `Запустить сейчас? (y/n)` — нажми `y` и Enter.

**Шаг 4.** Используй цифры для выбора функций в меню.

> Если хочешь запустить повторно:
> ```bash
> cd ~/gen_data && python generator.py

## ❓ Частые вопросы

**Q: Команда не работает, пишет `curl: command not found`**  
A: Установи curl: `pkg install curl` (Termux) или `sudo apt install curl` (Linux)

**Q: Пишет `python: command not found` при запуске**  
A: Попробуй `python3 generator.py` вместо `python generator.py`

**Q: Хочу обновить до последней версии**  
A: Запусти команду установки заново — она автоматически удалит старую версию и поставит новую

**Q: Как удалить?**  
A: `rm -rf ~/gen_data ~/install_gen.sh`
